# Contact Book (Python CLI)

This is a simple contact management system built using Python.

## Features

* Add Contact
* View Contacts
* Search Contact
* Update Contact
* Delete Contact
* Data stored using JSON

## Technologies Used

* Python
* JSON (for storage)

## How to Run

1. Run the Python file
2. Follow menu options in terminal

* Add GUI
